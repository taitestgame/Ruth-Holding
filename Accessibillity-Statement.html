We at Ruth Holding are dedicated to providing an accessible website for all users, including those with disabilities.

What web accessibility is

Web accessibility aims to ensure that people with disabilities can perceive, understand, navigate, and interact with the web, and that they can contribute to the web. It encompasses all disabilities that affect access to the web, including visual, auditory, physical, speech, cognitive, and neurological disabilities.

Accessibility adjustments on this site

Ruth Holding's website has been designed and developed in compliance with WCAG [2.0 / 2.1 / 2.2 - select relevant option] guidelines, and has been optimized for [A / AA / AAA - select relevant option] level of accessibility. We have implemented various features to enhance accessibility, including:

  • Keyboard navigation and screen reader compatibility
  • Descriptive alt text for images
  • High color contrast for improved readability
  • Limited use of motion and auto-play media
  • Transcripts and captions for audio and video content
